2019-03-11 14:00:00 Java

总结遇到的Java面试题

Java基础

  1. Java面向对象的特征

    • 封装、继承和多态

  2. 多态存在的三个必要条件

    • 要有继承
    • 要有重写
    • 父类引用指向子类对象

  3. 多态的应用场景

    • 同一个接口不同的实现方式

  4. 接口的意义

    • 接口有利于代码规范
    • 接口有利于代码的维护
    • 保证代码的安全与严密

  5. 接口与抽象类的区别

    • 接口是完全抽象的，不存在方法体，而抽象类可以同时包含抽象与非抽象方法
    • 类可以实现多个接口但是只能继承一个抽象类
    • 类可以不实现抽象类和接口声明的所有方法，在这种情况下，类必须是抽象类
    • 抽象类可以在不提供接口方法实现的情况下实现接口
    • 接口中声明的变量默认是final的，而抽象类可包含非final的变量
    • 接口中的方法都必须是public，抽象类则不然
    • 接口不可被实例化，抽象类也不可被实例化，但如果抽象类中包含main方法可以被调用
    • 接口不能有构造方法，抽象类可以

  6. &与&&的区别

    • &是位运算符，表示按位与运算，&&是逻辑运算符，表示逻辑与（and）
    • &&还具有短路的功能

  7. switch是否支持byte与long

    • 支持byte不支持long

  8. ==与eques的区别？equals与hashCode的区别

    • ”==”比较的是两个字符串对象的地址，equals()是比较的两个字符串的具体值。
    • equals方法主要是用来判断从表面上看或者从内容上看，2个对象是不是相等，hashCode相当于是一个对象的编码

  9. Object的公共方法

    • clone()
    • getClass()
    • toString()
    • finalize() 该方法用于释放资源
    • hashCode() 该方法用于哈希查找，可以减少在查找中使用equals的次数，重写了equals方法一般都要重写hashCode方法
    • wait() 调用该方法后当前线程进入睡眠状态
    • notify() 该方法唤醒在该对象上等待的某个线程
    • notifyAll() 该方法唤醒在该对象上等待的所有线程

  10. 堆和栈的区别

    • 在函数中定义的一些基本类型的变量和对象的引用变量都在函数的栈内存中分配
    • 堆内存用来存放有new创建的对象和数组
    • 在堆中产生一个数组或者对象之后，还可以在栈中定义一个特殊变量，让栈中的这个变量的取值等于数组或对象在堆内存中首地址，栈中的这个变量就变成了数组或者对象的引用变量，以后就可以在程序中使用栈中的引用变量来访问堆中的数组或者对象
    • 栈中的变量指向堆内存中的变量，这就是Java的指针

  11. 静态变量与实例变量的区别

    • 静态变量要在前加static关键字修饰
    • 实例变量属于某个对象的变量，必须创建实例才能被使用
    • 静态变量不属于某个实例对象，而是属于类，所以也称为类变量

  12. JVM性能调优

    • JVM将内存划分为New、Tenured(年老)、Perm(永久)，其中New和Perm属于堆内存
    • 线程池
    • 连接池
    • JVM启动参数：调整各代的内存比例和垃圾回收算法，提高吞吐量
    • 程序算法优化：改进程序逻辑算法提高性能
    • 减少使用全局对象和大对象

  13. 反射

    • Java反射就是在运行状态中，对于任意一个类，都能够知道这个类的所有属性和方法；对于任意一个类都能够调用它的任意方法和属性，并且能改变它的属性。
    • 代码可以在运行时装配，无需在组件之间进行源代码的链接，降低代码的耦合度，实现动态代理等

  14. static的作用和用法

    • 用于修饰成员变量，成员方法，而static修饰的成员变量称之为静态变量或类变量
    • static修饰的成员被所有的对象共享
    • static优先于对象存在，因为static的成员随着类的加载就已经存在
    • static修饰的数据是共享数据，对象中的存储的是特有数据

  15. JDK常见的包有哪些？

    • java.lang 包含一些java语言的核心类，如String、Math、Integer、System和Thread
    • java.util 包含一些使用的工具类
    • java.awt 抽象窗口工具包
    • java.applet 编制applet用到的包
    • java.net 包含执行与网络相关的操作的类
    • java.io 包含能提供多种输入输出功能的类
    • java.sql 包含数据库操作的类

JSP

  1. Servlet的生命周期
    • 加载和实例化
    • 初始化
    • 请求处理
    • 服务终止

Linux

  1. Linux常用的基本命令有哪些？
    • cd命令 切换目录
      • cd / 进入目录
      • cd ~ 进入home目录
      • cd .. 返回上级目录
      • cd !$ 把上个命令的参数作为cd使用
    • ls命令 查看文件目录信息
      • ls -a 列出目录所有文件，包含隐藏文件
      • ls -A 列出除…及…的其他文件
      • ls -r 反序排列
      • ls -t 以文件修改时间排列
      • ls -S 以文件大小排列
      • ls -l 除了文件名之外，还将文件的权限、所有者、文件大小等信息详细列出来
    • pwd命令 查看当前工作路径
    • mkdir命令 创建文件夹
      • mkdir -m 对新建目录设置存储权限
    • rm命令 删除文件
      • rm -i 删除前询问确认
      • rm -rf 删除改目录切包含子目录所有文件，并不用确认
    • mv命令 移动文件
    • cp命令 复制文件
    • cat命令
      • cat filename 一次展示整个文件
      • cat > filename 创建新文件
      • cat file1 file2 > file 合并文件
    • more命令 读取文件
    • less命令 less与more类似，但使用less可以随意浏览文件，而more仅能向前移动，却不能向后移动，而且less 在查看之前不会加载整个文件
