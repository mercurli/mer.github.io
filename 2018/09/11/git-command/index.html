git常用命令

2018-09-11 22:28:47 git

项目初始化

git init

创建本地代码仓库

提交项目版本

git add .

添加所有文件

git add <文件名>

添加单个文件

git commit -m "<消息内容>"

提交某个版本，-m 后面加上提交参数

绑定仓库和上传文件

git remote add origin <仓库名称>

关联到远程库

git pull --rebase origin master

获取远程库与本地同步合并

git push -u origin master

将文件上传到远程库（未绑定账号需要提供账号密码）
