Java面试题-3

2019-03-11

  1. Linux常用的基本命令有哪些？

    • cd命令 切换目录
      • cd / 进入目录
      • cd ~ 进入home目录
      • cd .. 返回上级目录
      • cd !$ 把上个命令的参数作为cd使用
    • ls命令 查看文件目录信息
      • ls -a 列出目录所有文件，包含隐藏文件
      • ls -A 列出除…及…的其他文件
      • ls -r 反序排列
      • ls -t 以文件修改时间排列
      • ls -S 以文件大小排列
      • ls -l 除了文件名之外，还将文件的权限、所有者、文件大小等信息详细列出来
    • pwd命令 查看当前工作路径
    • mkdir命令 创建文件夹
      • mkdir -m 对新建目录设置存储权限
    • rm命令 删除文件
      • rm -i 删除前询问确认
      • rm -rf 删除改目录切包含子目录所有文件，并不用确认
    • mv命令 移动文件
    • cp命令 复制文件
    • cat命令
      • cat filename 一次展示整个文件
      • cat > filename 创建新文件
      • cat file1 file2 > file 合并文件
    • more命令 读取文件
    • less命令 less与more类似，但使用less可以随意浏览文件，而more仅能向前移动，却不能向后移动，而且less 在查看之前不会加载整个文件

  2. JDK常见的包有哪些？

    • java.lang 包含一些java语言的核心类，如String、Math、Integer、System和Thread
    • java.util 包含一些使用的工具类
    • java.awt 抽象窗口工具包
    • java.applet 编制applet用到的包
    • java.net 包含执行与网络相关的操作的类
    • java.io 包含能提供多种输入输出功能的类
    • java.sql 包含数据库操作的类
